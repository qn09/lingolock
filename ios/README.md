# iOS Native App: Language Word of the Day (Lockscreen Widget)

This folder contains the complete, native iOS project source code written in Swift and SwiftUI. It supports iOS 16+ Lock Screen widgets and Home Screen widgets using WidgetKit.

## Directory Structure
- **Shared/**: Files shared between the Main App and the Widget.
  - [Word.swift](file:///home/quan/language-lockscreen-app/ios/Shared/Word.swift): The data model for vocabulary.
  - [VocabularyData.swift](file:///home/quan/language-lockscreen-app/ios/Shared/VocabularyData.swift): The offline vocabulary list and calendar-based deterministic daily word rotation logic.
  - [AppSettings.swift](file:///home/quan/language-lockscreen-app/ios/Shared/AppSettings.swift): Manages settings stored in shared `UserDefaults` using App Groups.
- **WordOfTheDay/**: The main application views and application entry point.
  - [WordOfTheDayApp.swift](file:///home/quan/language-lockscreen-app/ios/WordOfTheDay/WordOfTheDayApp.swift): Main application file.
  - **Views/**:
    - [MainDashboardView.swift](file:///home/quan/language-lockscreen-app/ios/WordOfTheDay/Views/MainDashboardView.swift): Displays the current word card, word translation, part of speech, meanings, examples, and speaks pronunciation using `AVSpeechSynthesizer`.
    - [HistoryView.swift](file:///home/quan/language-lockscreen-app/ios/WordOfTheDay/Views/HistoryView.swift): Reviews history of previously shown words with a detail search interface.
    - [SettingsView.swift](file:///home/quan/language-lockscreen-app/ios/WordOfTheDay/Views/SettingsView.swift): Modifies targets and displays how-to instructions.
- **WordOfTheDayWidget/**: The Lock Screen/Home Screen Widget Extension.
  - [WordOfTheDayWidget.swift](file:///home/quan/language-lockscreen-app/ios/WordOfTheDayWidget/WordOfTheDayWidget.swift): The timeline provider and layout views for `.accessoryInline`, `.accessoryRectangular`, `.accessoryCircular`, `.systemSmall`, and `.systemMedium` widgets.

## How to Set Up in Xcode on a Mac

To build and run this app on your iPhone:

### Step 1: Create the Xcode Project
1. Open Xcode on a Mac.
2. Select **File > New > Project...**.
3. Choose **iOS > App** and click **Next**.
4. Name the product **WordOfTheDay**.
5. Set **Interface** to **SwiftUI** and **Language** to **Swift**, then click **Create**.

### Step 2: Add the Widget Extension Target
1. In Xcode, select **File > New > Target...**.
2. Select **iOS > Widget Extension** and click **Next**.
3. Name the target **WordOfTheDayWidget**.
4. **UNCHECK** the "Include Configuration Intent" checkbox (we are using a static timeline) and click **Finish**.
5. When prompted, click **Activate** to activate the widget's build scheme.

### Step 3: Copy the Source Files
1. Delete the default files generated by Xcode (`ContentView.swift`, `WordOfTheDayApp.swift`, and `WordOfTheDayWidget.swift`).
2. Drag and drop the directories from this repository into Xcode's Project Navigator:
   - Drag `Shared/` into the root of the project. In the pop-up, make sure **both** `WordOfTheDay` and `WordOfTheDayWidget` targets are checked in **"Target Membership"**.
   - Drag `WordOfTheDay/` into the project (ensure target membership is only `WordOfTheDay`).
   - Drag `WordOfTheDayWidget/` into the project (ensure target membership is only `WordOfTheDayWidget`).

### Step 4: Configure App Groups (Required for Widget Syncing)
Since the Widget and the Main App run in separate sandboxes, they must communicate via a shared container called **App Groups**.
1. Select the top-level **WordOfTheDay** project node in Xcode.
2. Under the **Signing & Capabilities** tab:
   - Click **+ Capability** in the top-left corner.
   - Search for **App Groups** and double-click to add it.
   - Click the `+` button in the App Groups list and add a group named: `group.com.language.wordoftheday` (or `group.yourdomain.WordOfTheDay`). Make sure it is checked.
3. Now switch to the **WordOfTheDayWidget** target under the Targets list.
   - Perform the same steps: Add the **App Groups** capability.
   - Check the **same** App Group name (`group.com.language.wordoftheday`).
4. If you used a custom domain name (e.g. `group.yourdomain.WordOfTheDay`), open [AppSettings.swift](file:///home/quan/language-lockscreen-app/ios/Shared/AppSettings.swift) and [WordOfTheDayWidget.swift](file:///home/quan/language-lockscreen-app/ios/WordOfTheDayWidget/WordOfTheDayWidget.swift) and update `appGroupId` value to match your custom group name.

### Step 5: Build and Run!
1. Connect your iPhone via USB, or select an iOS Simulator.
2. Hit **Cmd + R** (or click the Play button) to build and run the app.
3. Follow the steps in the App Settings to customize your iPhone Lock Screen and place your new learning widget!
